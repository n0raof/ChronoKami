# ChronoKami: Программный комплекс адаптивного тайм-менеджмента и оптимизации академической успеваемости

## 📌 Обзор проекта
**ChronoKami** — это интеллектуальный программный комплекс на базе платформы Telegram, предназначенный для управления временем, минимизации деструктивных паттернов прокрастинации и систематизации учебного процесса студентов. 

---

## 🔬 Научно-методологическая база
В основу проекта заложена интеграция двух ключевых концепций:
1. **Матрица Эйзенхауэра:** Все поступающие задачи декомпозируются по осям Важности и Срочности. Focus на Квадранте B (*Важно, но несрочно*).
2. **Поведенческий трекинг:** Модуль формирования привычек использует циклическую модель закрепления нейронных связей

---

### 1. Требования к окружению
Для сборки и запуска комплекса необходим интерпретатор Python версии 3.10 или выше, а также менеджер пакетов `pip`
